##目录说明
```
index.js 	项目启动入口文件
common      为我们自己写的通用模块
 - MyJdbc   封装后的JDBC模块
            包含MyPool类(连接池),MyConnection类,集成了Execute,Query,Commit,Rollback,QueryLimit等好的方法,支持参数绑定,避免SQL注入
 - - Model  数据表转Object模型类,集成了insert,select,selectOne,update,delete等方法
 - - SqlHelper 方便转换为Sql的对象
 - xlsx     Excel操作
 - MyUtil   简单的常用方法封装
 - email    发送邮件
 - request  http/https请求简单封装
data		网站等参数目录
 - conf.js 	网站设置及数据库参数
 - conf_portal.js   门户授权参数配置
docs        文档``
 - README.md
 - 目录说明.md
 - 虚拟机说明.md
resource	html前端页面
router		路由文件目录
 - index.js 	路由表入口
 - rest 对外提供API统一入口,访问地址:http://127.0.0.1/router/{db}/rest
interface   接口中心
 - common 统一接口调用,进行标准化输入,输出的转换工作
 - top	淘宝接口目录
 - jos  京东接口
 - vop    唯品会接口 
 - oms
 - youzan 有赞接口
 - - api 原生接口封装成方法
 - - apiExt 拓展或优化参数和输出,及记录到mongoDb (比如:getList,syncList,findList)
 - - convert 将输出结果进行统一化处理,由common里调用
service
 - db
 - - web.js	数据库统一连接
 - model		通用数据访问层目录
 - runsa 浪沙接口
 - event 数据操作事件定义 (如:订单发货前,订单发货后,以及异步事件...)
 - vop 唯品会服务
 - task 计划任务
 - stock 库存服务
 - interface 含数据库连接对外接口,并拓展增量同步到数据表的方法 (例:syncList2)
 - - index.js 对外调用统一转发到/interface/common/index.js
 - - message 消息服务,用于接收外部系统的推送消息
 - - task 和接口有关的计划任务
```
